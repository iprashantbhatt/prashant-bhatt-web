# **App Name**: PrashantOS

## Core Features:

- 3D Profile Showcase: Displays a centered, rounded, and slightly extruded 3D profile image of Prashant Bhatt as the visual focal point, with smooth hover animations.
- Apple-style Home Screen UI: Implements an Apple iPad/iPhone-like home screen UI with glassmorphism, soft blur, and depth effects. Includes a real-time clock with timezone auto-detection, battery percentage indicator (visual only), and subtle signal/Wi-Fi icons.
- 3D Floating App Icons with Dock Layout: Features 3D, slightly floating app icons in a dock-style layout. Each icon links to Prashant Bhatt's social media (Instagram, LinkedIn, GitHub, X/Twitter), portfolio/blog, and email/contact.
- Parallax 3D Transition: Enables a parallax 3D scrolling/swiping effect to transition between sections (homepage and developer work page), with a smooth iOS-like slide-up animation for the next page.
- AI-Powered Personalization Tool: Generative AI tool suggests content for the webpage, considering visitor information such as the referral link used to arrive at the webpage.
- AI Project Display: Dynamically shows Prashant Bhattâ€™s developer work including AI Agents, Web Apps, and ongoing projects with 3D glass card-style project cards and floating hover animations.
- Responsive Design Adaptation: Adapts the layout for different devices: desktop (iPad-like centered layout) and mobile (iPhone-like full-screen experience) with touch-friendly gestures.
- PrashantGPT AI Chatbot: Gemini-powered AI chatbot for user interaction and assistance.

## Style Guidelines:

- Primary color: Light blue (#B3DDF2) for a calm and professional feel.
- Background color: Very light blue (#F0F8FF) to create a clean, airy backdrop, in keeping with the light scheme.
- Accent color: Soft Lavender (#D0B3F2) for highlights and interactive elements.
- Body and headline font: 'Inter' for a modern, machined, objective, neutral look.
- Apple-style icons, designed with soft shadows, reflections, and depth to give a 3D floating effect.
- Centered layout on desktop to mimic an iPad screen, transitioning to a full-screen layout on mobile devices, similar to an iPhone.
- Smooth, physics-based animations for hover effects, transitions, and scrolling to enhance the premium feel. Subtle light reflections and motion to bring elements to life.
